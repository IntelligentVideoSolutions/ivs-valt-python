# ivs-valt-python

A Python client library for interacting with the **Intelligent Video Solutions (IVS) VALT** REST API. This library simplifies the management of video assets, room controls, and administrative tasks within the VALT ecosystem.

## Features

* **Room Control:** Start, stop, pause, and resume recordings programmatically.
* **Asset Management:** List, filter, and manage recorded videos and markers.
* **Administrative Tools:** Manage users, user groups, cameras, and recording templates.
* **Scheduling:** Create and update recording schedules via the VALT scheduler.
